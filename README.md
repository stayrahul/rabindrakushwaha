A sleek, high-performance, single-page portfolio designed for a strategic visionary and entrepreneur. This project utilizes a modern "glassmorphism" aesthetic, asymmetric bento-grid layouts, and smooth scroll animations to create an elite digital presence.

## ✨ Features

* **Modern UI/UX:** Dark-mode native design with a sophisticated Zinc and Amber color palette.
* **Glassmorphism:** Premium frosted-glass card components with dynamic hover states.
* **Scroll Animations:** Elements reveal smoothly as the user scrolls down the page using the native `IntersectionObserver` API.
* **3D Interactions:** Desktop cards feature subtle 3D tilt effects powered by Vanilla-Tilt.js.
* **Fully Responsive:** Carefully optimized for mobile, tablet, and desktop viewing with fluid typography and a custom mobile navigation menu.
* **Zero Build Step:** Runs purely on CDN links—no Node.js or `npm install` required.

## 🛠️ Built With

* **[HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML):** Semantic structure.
* **[Tailwind CSS (v3)](https://tailwindcss.com/):** Utility-first styling via CDN for rapid UI development and custom themes.
* **[Alpine.js](https://alpinejs.dev/):** Lightweight JavaScript framework used for managing the mobile menu state and scroll progress bar.
* **[Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/):** Smooth 3D tilt interactions on hover.

## 🚀 Getting Started

Since this is a completely static website with no build tools, running it locally is incredibly simple.

1. **Clone the repository:**
       git clone https://github.com/stayrahul/rabindra-portfolio.git
   
2. **Navigate to the directory:**
       cd rabindra-portfolio
   
3. **Run the site:**
   Simply double-click the `index.html` file to open it in your default web browser. Alternatively, you can use a tool like VS Code's "Live Server" extension for hot-reloading during development.

## 🌍 Deployment

This project is perfectly optimized for immediate deployment on **Vercel** or **GitHub Pages**.

**To deploy on Vercel:**
1. Create a free account on [Vercel](https://vercel.com).
2. Connect your GitHub account and import this repository.
3. Leave all build settings blank (Framework Preset: Other).
4. Click **Deploy**. Your site will be live globally in seconds.
